# User Guide

This guide walks through every page in Priorities.ai in the order they appear in the navigation sidebar. Use it as a reference for what each page does, what actions are available, and how it fits into the overall workflow.

## How the app is organized

The sidebar groups pages into functional areas. You move through them roughly in this sequence when running a full prioritization effort:

| Section | Purpose |
|---|---|
| **Dashboard** | Strategic awareness at a glance |
| **Tasks** | Your personal work queue |
| **Search** | Find anything across the workspace |
| **Strategy** | Define the strategic context that guides prioritization |
| **Clarity Tools** | Structured thinking — questions, outcomes, back plans |
| **Intake** | Collect and triage incoming ideas |
| **Items** | Create, enrich, and manage the work candidates |
| **Relationships** | Model dependencies, clusters, and hierarchies between items |
| **Cycles** | Configure and run a prioritization effort |
| **Sessions** | Facilitated ranking events inside a cycle |
| **Catchball & Governance** | Proposals, ratification, and commitment tracking |
| **Priority Lists & Plans** | Published results and plans of record |
| **Reports** | Analysis and audit artifacts |
| **Settings** | Workspace configuration, access control, and integrations |

## Navigation basics

- **Sidebar** — The left-hand rail is the primary navigation. Some sections are collapsible. Sections that require a module or feature flag only appear when enabled.
- **Command palette** — Press `⌘ K` (Mac) or `Ctrl K` (Windows/Linux) to jump to any page or action without using the mouse.
- **Pinned cycle** — When a cycle is pinned, the sidebar gains cycle-specific shortcuts (Workspace, Tracks, Sessions) that point directly into that cycle's context.
- **Breadcrumbs** — A dual breadcrumb bar appears when you are inside a cycle, track, or session context, showing where you are and letting you navigate up the hierarchy.
- **Notifications** — The bell icon in the top bar opens your notification center.
